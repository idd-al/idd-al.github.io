Overview

Interaction Design is an interdisciplinary field merging design skills such as user experience, interface development, and graphic design with programming to create meaningful experiences between the user, digital, and physical platforms. This ever-evolving field envisions new ways of human-computer interaction within systems, services, products, and spaces across multiple industries.
